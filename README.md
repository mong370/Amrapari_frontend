# DWAR

**DWAR** is a modern frontend web application built with Vue and Vite.
The word **“Dwar”** means *door or gateway*, representing the entry point for users to access the system.

---

## 🎯 Project Purpose

DWAR acts as the **user entry portal** for accessing different services of the system.
It provides authentication, dashboards, and user interfaces for interacting with backend microservices.

---

## 🚀 Technology Stack

* Vue.js
* Vite
* Tailwind CSS
* Vue Router
* Pinia (State Management)

---

## 📂 Project Structure

```bash
src
├── assets/           # Images, fonts, styles
├── components/       # Reusable UI components
├── views/            # Page-level components
├── router/           # Application routing
├── stores/           # State management
├── services/         # API communication
├── utils/            # Helper functions
├── layouts/          # Layout components
├── plugins/          # Plugin configuration
├── config/           # Application configuration
├── App.vue
└── main.js
```

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://gitlab.quantum.org.bd/amrapari/dwar.git
cd dwar
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

The application will run on:

```bash
http://localhost:5000
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
